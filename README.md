# 逻辑猫事务所 · Logic Cat Office V2

React + Vite 品牌官网第二版。

## 本地运行

```bash
npm install
npm run dev
```

## 生产构建

```bash
npm run build
```

## 本版内容

- 品牌首页 Hero
- 事务所说明
- 员工档案室
- 明星员工展览板
- 今日案件
- 随机值班员工
- 事务所实时状态彩蛋
- 档案馆入口
- 档案保管员 L
- 收尾页

## 更新线上网站

将压缩包中的全部项目文件覆盖上传到 GitHub 仓库根目录。不要上传 `node_modules`。
Vercel 会自动重新部署。
